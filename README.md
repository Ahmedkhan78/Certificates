# My Certifications Portfolio

A showcase of my completed certifications in **web development, programming, data analysis, and other technical skills**. Each certificate represents hands-on experience and mastery in its respective field.

---

## Features

- Responsive **FilterBar** to view certificates by platform.
- Interactive **certificate cards** with hover effects.
- **Light rays background** for a cool visual effect.
- Click on a certificate image to **preview it in full size**.
- Fully responsive on desktop, tablet, and mobile devices.

---

## Technologies Used

- **React** – Frontend framework
- **JavaScript (ES6+)**
- **CSS3** – Styling & responsive design
- **WebGL / OGL** – Light rays effect
- **JSON** – Certificates data
